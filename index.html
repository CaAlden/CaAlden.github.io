Weather Cube

The Weather Cube was a joint project that I built with my friend. The cube is an Arduino with an Adafruit LED and a Bluetooth receiver. A python script running on the master device pulls weather data from Weather Underground and calculates a color to send to the LED cube. The LED cube waits for color code input over Bluetooth and updates its color when it receives that input. The cube could be repurposed for any application that wants to control its colors.

  • Date: Fall 2015
  • Topics: Bluetooth, Weather API, Arduino

Podric

Podric is my personal podcast organizer. Podric is configured with a list of podcast RSS feeds. It pulls the data and presents and Object Oriented API. The python library portion of this project is finished, and the command line interface works as a proof of concept. Moving forward, I am going to build a case and implement the library on a Rasberry Pi.
